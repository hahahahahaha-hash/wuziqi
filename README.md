# ⚪⚫ 极简五子棋 (Gomoku Online)

这是一个使用原生 **HTML5 Canvas**、**CSS3** 和 **JavaScript** 构建的经典双人对弈五子棋游戏。它不仅是一个游戏，更是一个展示 DOM 交互与坐标算法逻辑的练手项目。

## 🌟 项目亮点

* **智能落点**：精准的坐标捕获，点击网格交叉点即可落子。
* **胜负判定**：内置高效的扫描算法，实时检测横、竖、左斜、右斜四个方向。
* **悔棋功能**：支持撤销操作，给“手滑”一个机会。
* **视觉反馈**：丝滑的落子动效与当前回合提示。

## 📂 文件结构

```text
├── index.html   # 页面结构与 Canvas 容器
├── style.css    # 棋盘木纹质感与 UI 样式
└── script.js    # 核心游戏逻辑（棋盘渲染、落子算法、胜负检查）

```

## 🕹️ 如何玩耍

1. **启动**：在浏览器中打开 `index.html`。
2. **对弈**：黑棋先行，点击棋盘上的交叉点落子。
3. **规则**：任何一方在横向、纵向或斜向先连成 **5 颗** 相同颜色的棋子即获胜。
4. **重置**：刷新即可清空棋盘。

## 🧠 核心算法解析

五子棋最核心的部分在于**胜负判定逻辑**。本项目采用的是“中心扩散扫描法”：

> 每当一颗棋子落下时，以该坐标为中心 $(x, y)$，分别向 8 个方向进行延伸计数。若在任一轴线上（如水平轴）的连续同色棋子总数 $Count \ge 5$，则判定当前玩家胜利。

## 🛠️ 自定义配置

你可以在 `script.js` 中轻松修改以下参数：

* **棋盘格数**：默认为 15x15（标准棋盘）。
* **棋盘颜色**：修改 Canvas 绘图 API 中的 `strokeStyle`。
